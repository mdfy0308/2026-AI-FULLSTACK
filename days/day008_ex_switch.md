### ■2. Todo1: CSS 내부적용 선택자

선택자
- 일반 태그 선택자 : p
- 아이디 선택자 : #id
- 클래스 선택자 : .class
- 가상선택자 : hover, first-child, last-child
    :hover 마우스를 올렸을 때
    :first-child 부모 안에 있는 첫번째 자식요소
    :last-child 부모 안에 있는 마지막 자식요소 
    :nth-child(n) 부모 안에 있는 n번째 자식요소
    감싸는 박스(부모)를 기준으로 순서를 정함
