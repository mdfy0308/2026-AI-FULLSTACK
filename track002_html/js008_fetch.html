Ajax (Asynchronous JavaScript and XML)

비동기 통신 : 페이지 전체를 새로고침 안해도 서버와 데이터를 주고 받을 수 있음

데이터 포맷 : xml, json, csv

Ajax-text fetch() 로컬텍스트파일 가져오기

Q1. Ajax-text (fetch) 

        1. fetch()로 로컬 텍스트 파일 가져오기
            경로: ./data/text2.txt
            예시 내용:
            사과/1000 
        2. 응답을 .text()로 받아서 줄 단위로 나누기
        3. 각 줄을 / 로 split하여 name, price 출력하기
